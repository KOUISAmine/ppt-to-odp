# PPT to ODP [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

>PPT to ODP is an online tool that converts PPT/PPTX (PowerPoint Presentation) files into ODP (OpenDocument Presentation) files.

Try demo: https://tools.waytolearnx.com/ppt-to-odp
